# Покер для двоих
Простая версия покера, в которую можно играть вдвоем.

### Основные действия
- ****Сбросить** (**Fold**) ** — сказать "пас" и завершить раунд
- ****Уравнять** (**Call**) ** — поддержать текущую ставку
- ****Поднять** (**Raise**) ** — увеличить ставку

### Процесс игры
1. **Раздача** — ПРЕФЛОП — каждый получает по 2 карты
2. **Три первых карты на столе** — ФЛОП
3. **Четвёртая общая карта** — ТЁРН
4. **Пятая общая карта** — РИВЕР
5. **Вскрытие карт** — ШОУДАУН
На каждой из первых четырёх итераций происходят торги.
Если до окончания игры один из игроков сбросит карты (fold), то второй игрок автоматически становится победителем.
Иначе карты вскрываются и игроки сами оценивают, кто победил.
В конце игры происходит проверка на её валидность

### Комбинации (от сильной к слабой)
1. **Флеш-рояль** — 10, J, Q, K, A одной масти
2. **Стрит-флеш** — 5 карт одной масти по порядку
3. **Каре** — 4 карты одного значения
4. **Фулл-хаус** — тройка + пара
5. **Флеш** — 5 карт одной масти
6. **Стрит** — 5 карт по порядку
7. **Тройка** — 3 карты одного значения
8. **Две пары** — две разные пары
9. **Пара** — 2 карты одного значения
10. **Старшая карта** — самая высокая карта на руках

## Инструкция к запуску
1. Скачайте архив и разархивируйте на своём компьютере
2. Найдите папку dist и откройте её
3. Двойным нажатием запустите launcher.exe и добро пожаловать в консоль
4. Если вы подключаетесь первым, выберите роль "Хост", вторым — "Клиент"
5. Хост получит свой IP-адрес, его надо указать при подключении Клиента
6. В бой! Пусть скучное время пройдёт с небольшим азартом

## Кто побеждает?
Победителем становится тот, кто заставил соперника сбросить карты или собрал более сильную комбинацию при вскрытии
